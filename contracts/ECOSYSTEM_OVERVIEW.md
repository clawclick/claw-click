## 🎯 WHAT IS CLAWCLICK?

ClawClick is a **Uniswap V4 hook-based token launch platform** that implements a two-phase progressive tax model. It enables fair token launches with anti-bot protection that gradually removes as tokens gain organic traction.

### Key Innovation
- **Phase 1:** Protected launch with progressive tax (50% → 1%)
- **Phase 2:** Graduated "normal" token after 4x MCAP growth
- **100% protocol-owned liquidity** (locked forever, never extractable)
- **70/30 fee split** (creator/platform)
- **Deterministic pricing** from target market cap

## 🔄 TOKEN LAUNCH FLOW

### User Perspective:
1. User calls `Factory.createLaunch(name, symbol, targetMCAP, beneficiary)`
2. Factory deploys token → initializes pool → locks liquidity
3. Token immediately tradable with progressive tax
4. Tax decays as market cap grows
5. After 4x MCAP growth sustained 1hr → graduates
6. Post-graduation: normal token with 1% LP fee

### Under the Hood:
1. **Factory** deploys **ClawclickToken** (1B supply)
2. **Factory** calculates sqrtPrice from target MCAP
3. **Factory** initializes Uniswap V4 pool with **Hook** attached
4. **Factory** adds full-range liquidity (100% token, 0 ETH)
5. **Factory** transfers LP NFT to **LPLocker** (permanent lock)
6. **Hook** registered for pool, tracks stats
7. Every swap → **Hook** applies progressive tax → collects fees
8. Fees split 70/30 → beneficiary/platform
9. After 4x MCAP → **Hook** detects graduation
10. Post-graduation → Hook no-ops, pool fee = 1%

## 💰 ECONOMIC MODEL

### Tax Tiers (Target MCAP → Starting Tax)
- **1 ETH:** 50% starting tax
- **2 ETH:** 40% starting tax
- **5 ETH:** 20% starting tax
- **10 ETH:** 5% starting tax

### Tax Decay (Logarithmic)
```
Tax = BaseTax / (2^doublings)
Floor = 1% (never 0%)

Example (1 ETH start):
  1 ETH → 50.0%
  2 ETH → 25.0%
  4 ETH → 12.5%
  8 ETH → 6.25%
 16 ETH → 1.0% (floor)
```

### Fee Distribution
- **70%** → Token beneficiary (creator)
- **30%** → Platform treasury

### Transaction Limits
- Scale dynamically with MCAP
- **1 ETH MCAP:** 0.3% max tx, 0.3% max wallet
- **10 ETH MCAP:** 0.1% max tx, 0.1% max wallet
- **Post-graduation:** No limits

## 🎓 KEY CONCEPTS

### Uniswap V4 Hooks
Hooks are smart contracts that can execute custom logic before/after pool operations (swaps, liquidity changes). ClawClick uses a hook to implement progressive tax and transaction limits WITHOUT modifying the token contract itself.

### CREATE2 Deployment
Hooks must be deployed using CREATE2 with a specific salt to ensure the resulting address has the required permission flags encoded in its leading bytes. Use `HookMiner.sol` to find a valid salt.

### Protocol-Owned Liquidity
LP positions are minted as ERC-721 NFTs. ClawClick locks these NFTs in the `ClawclickLPLocker` contract forever. They can NEVER be withdrawn to an EOA, ensuring permanent liquidity. Post-graduation rebalancing is allowed but still maintains protocol ownership.

### Graduation
When a token's MCAP reaches 4x its starting MCAP and sustains for 1 hour, it "graduates". At this point, the hook stops applying tax, the pool fee switches to 1%, and transaction limits are removed. This is irreversible.
