# Claw.Click Website

Professional black and white website for Claw.Click trading infrastructure platform.

## Pages

### 🏠 Homepage (/)
- Hero section with animated logo and terminal stats
- Value proposition with quoted statement
- Sliding carousel of 36+ integrated APIs
- Professional black/white design

### 📚 API Documentation (/api)  
- Professional API docs with syntax highlighting
- Comprehensive endpoint documentation
- Integration status dashboard
- Code examples with curl/Python
- Authentication and rate limiting info

### 🛒 Strategy Marketplace (/app)
- Trading strategy marketplace
- Strategy cards with metrics (win rate, APY, volume, chain)
- Risk assessment and filtering
- Professional strategy deployment interface

## Design System

- **Colors**: Pure black (#000) and white (#fff) for professional infra vibe
- **Syntax Colors**: Terminal-inspired highlighting (red, green, yellow, blue)
- **Typography**: Inter for text, JetBrains Mono for code/stats
- **Layout**: Clean sections with proper spacing and responsive design

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview build
npm run preview
```
